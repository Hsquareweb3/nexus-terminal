# LuxTrade — Premium Crypto Trading Dashboard

A distinctive cryptocurrency trading terminal with luxury fintech aesthetics.

## Framework

Static HTML with Vanilla JavaScript and Chart.js

## Preview

Configured for immediate preview. The system handles installation and startup automatically.

## Theme

**Luxury Fintech Dark** — Deep charcoal blacks (#0a0a0f, #0f0f1a, #1a1a2e) with electric blue accents (#3b82f6) and subtle gold highlights. Features layered shadows, subtle grain texture overlay, asymmetric accent bars, and micro-motion on hovers and live data updates.

## Typography

- **Display/UI**: Sora (clean, modern sans-serif with excellent legibility)
- **Data/Mono**: JetBrains Mono (technical, precise, optimized for numbers)

## Features

- **Left Sidebar**: Coin watchlist with live sparkline mini-charts for 6 cryptocurrencies
- **Main Chart**: Interactive candlestick-style price chart with gradient fill
- **Order Book**: Real-time bid/ask spread with depth visualization bars
- **Recent Trades**: Live trade feed with buy/sell indicators
- **Top Navigation**: Portfolio value display with 24h change tracking
- **Live Updates**: Simulated real-time price updates and trade flow

## Design Details

- Grain texture overlay for premium depth
- Gradient backgrounds with blur effects
- Custom scrollbars with blue gradient
- Hover micro-interactions (lift, glow, translate)
- Asymmetric accent bars for visual interest
- Layered shadow system for depth hierarchy
- Tabular number formatting for price displays
